# テレアポ CRM（個人用）

電話営業（テレアポ）の顧客管理を、一人で・無料で回すための軽量 Web アプリです。

- **架電ステータス管理**（未着手 / 不在・再コール / 見込みあり / アポ獲得 / NG）
- **次回架電日とリマインド**（「今日かける」「期限切れ」をワンクリック表示）
- **通話メモ・架電履歴**（1顧客に何件でも履歴が積み上がる）
- **CSV 取込 / 書き出し**（既存リストの読み込み・バックアップ）

## 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | Next.js 15（App Router / TypeScript） |
| データベース | SQLite（`data/teleapo.db` というファイル1個） |
| UI | プレーン CSS（依存なし） |
| 認証 | なし（自分ひとり用） |

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで <http://localhost:3000> を開きます。
初回アクセス時に `data/teleapo.db` が自動作成されます。

## 使い方

1. **顧客一覧**（トップ画面）
   - 検索・ステータス絞り込み・「今日かける／期限切れ」フィルタ
   - 「新規追加」で顧客を登録
   - 「CSV取込」で既存リスト（会社名・電話・担当者の列）を一括登録
   - 「CSV書き出し」で全顧客をダウンロード
2. **顧客詳細（架電画面）**：一覧の行をクリック
   - 架電結果（ステータス）・メモ・次回架電日を記録 → 履歴に1件積まれる
   - 基本情報の編集・削除

## データの保存場所

すべて `data/teleapo.db`（SQLite ファイル）に保存されます。
このファイルをコピーすればバックアップ・他PCへの移行ができます（`.gitignore` 済み）。
